1. Define what a function is in your own words?
A function is a reusable block of code that performs a specific task.
<br>

2. What key word is used to define a function?
The key word used to define a function is "def".
<br>

3. What does a return statement do?
A return statement stops the execution of a function and sends a specific value back to the part of the program that called it
<br>

4. What is the difference between arguments and parameters?
Arguments are the actual values or data you pass to a function when you call the function, 
while parameters are the variables placeholders for the function.
